🌈 Gradient System

Subtle gradients for section backgrounds, adding depth without distraction. Apply at 10-15% opacity for a ghostly effect that adds depth without distracting from content.

Purple Nebula

A dreamy purple gradient for creative sections. Uses adjacent purple hues for a seamless ethereal effect. Apply to creative workflow areas, music composition tools, and inspiration sections.

background: linear-gradient(135deg, #8b5cf6 0%, #7c3aed 50%, #6d28d9 100%);

Amber Glow

🎯 Color Usage Guidelines

🎵 Primary Purple

Guides users to key actions - CTAs, navigation, and primary workflows. Its vibrancy creates focal points without overwhelming. Use for: Main buttons, active nav items, primary brand elements.

⚡ Electric Blue

Indicates interactive elements, active states, and progress indicators. Its calming effect promotes focused work sessions. Use for: Links, progress bars, secondary actions.

🔥 Amber Gold

Highlights premium features, warnings, and value propositions. Used sparingly (≤10% of UI) to maintain impact. Use for: Premium badges, important alerts, upgrade prompts.

🌱 Emerald Green

Signals success, completion, and positive outcomes to reinforce user achievements. Use for: Success messages, checkmarks, completion states.

🎵 Design Philosophy & Rationale

Our color palette resonates with the creative energy of the music industry while maintaining the professionalism expected from a SaaS product. Each color represents core values for music creators:

Purple represents creativity and innovation - historically associated with royalty and artistry, positioning our brand as premium and visionary.
Electric Blue brings technological edge - symbolizing the digital transformation of music creation and collaboration.
Amber Gold adds warmth and energy - reflecting the passion and emotion inherent in music production.
Emerald Green provides balance - representing growth, harmony, and successful outcomes for creators.

💻 Implementation Guidelines

The palette follows a systematic approach with primary, secondary, and neutral colors clearly defined for consistent implementation across all platforms.

CSS Variables

Use these CSS custom properties for consistent theming and easy maintenance:

:root { /* Primary Purple */ --color-primary-500: #8b5cf6; --color-primary-600: #7c3aed; --color-primary-700: #6d28d9; /* Electric Blue */ --color-secondary-500: #06b6d4; --color-secondary-600: #0891b2; --color-secondary-700: #0e7490; /* Amber Gold */ --color-accent-500: #f59e0b; --color-accent-600: #f97316; --color-accent-700: #ea580c; /* Emerald Green */ --color-success-500: #10b981; --color-success-600: #059669; --color-success-700: #047857; /* Neutrals */ --color-gray-800: #1f2937; --color-gray-500: #6b7280; --color-gray-100: #f3f4f6; /* Gradients */ --gradient-purple-nebula: linear-gradient(135deg, #8b5cf6 0%, #7c3aed 50%, #6d28d9 100%); --gradient-amber-glow: linear-gradient(135deg, #f59e0b 0%, #f97316 50%, #ea580c 100%); --gradient-electric-surge: linear-gradient(135deg, #06b6d4 0%, #0891b2 50%, #0e7490 100%); --gradient-emerald-harmony: linear-gradient(135deg, #10b981 0%, #059669 50%, #047857 100%); }

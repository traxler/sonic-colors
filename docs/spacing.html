üéµ Spacing Rationale

Our 8-point grid system creates visual harmony and consistency across all interface elements.

Component Spacing

Use smaller spacing values (3xs, 2xs, xs) within components for tight relationships between related elements. Larger values (sm, md, lg) separate distinct sections or create breathing room.

Layout Hierarchy

Increase spacing as you move up the hierarchy. Text elements use 3xs-md, components use md-xl, and major sections use 2xl-4xl. This creates clear visual relationships and improves content scanning.

Responsive Considerations

On mobile devices, consider reducing spacing by one size (lg becomes md) to maximize content area while maintaining proportional relationships. Always test spacing on actual devices.
